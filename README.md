## [JS-Fastify-Blog](https://github.com/hexlet-components/js-fastify-blog) on docker

### Requirements
- docker
- docker compose
- make

### Setup
Clone repository, then run command  
```make generate-env``` for local containers running

### Run
- ```make build```
- ```make run```
